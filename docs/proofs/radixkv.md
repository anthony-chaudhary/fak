## Closures (witness pass 2026-06-20, commit `3cb8ff9`)

Each obligation marked OPEN above was discharged by a new zero-dependency (stdlib `testing`/`testing/quick`) metamorphic/round-trip/invariant test that ASSERTS the property against an independently recomputed reference. Verified by `go test -count=1 ./internal/...` (45 packages green, 0 failures).

- **refcount-conservation** → ✅ PROVEN by `TestRefcountConservationCycleNetsZero, TestRemovedNodeUnreachableNeverMatched`. Two tests close this jointly. (a) Lease conservation: over a mixed workload (cold, shared-prefix-split, exact-re-request/empty-suffix, disjoint) of 6 concurrent in-flight requests, Sigma node.refs (root INCLUDED) is asserted at every step: exactly +1 after each Lookup, UNCHANGED after each Insert (a lease handoff, not a new lease), equal to the live-request count while all 6 in flight, decremented to track each Done in a fixed-seed-shuffled (0x5eed) release order, =0 once all Done, and still =0 after an extra Done per node (refs floored at 0, no leak, never negative). KEY non-obvious mechanism the test made explicit: a COLD request leases the ROOT boundary (root.refs++) which Insert then hands off to the new leaf, so sumRefs must include the root or it under-counts an in-flight cold lease. (b) Dangling-node: a node removed by LRU eviction AND by EvictNode is asserted unreachable from root (children-map walk), its parent slot gone, MatchLen==0, and t.tokens reconciled against the live Sigma edge lengths (Stats().Tokens) so the counter never leaks on removal. All PASS.
- **lru-leaf-evicted-hot-retained** → ✅ PROVEN by `TestLRUEvictsOldestRetainsHotAndLeased, TestLRUUpwardCollapse`. Two tests close this. TestLRUEvictsOldestRetainsHotAndLeased: under budget=20 with three 10-token requests, (a) HOT retention: re-Looking-up the oldest (a) freshens its lastUsed, so when c overflows the budget the victim is b (the now-oldest UNLOCKED leaf), not a; asserts a and c retained, b evicted. (b) LEASED retention: in a second tree the oldest (x) is held leased (refs>0, never Done), so eviction skips it and removes the oldest UNLOCKED (y); asserts x and z retained, y evicted. TestLRUUpwardCollapse: tree root->preamble(8)->{tailA,tailB} at exactly budget=14; serving a disjoint 14-token request forces evictToBudget to remove both tails AND then the preamble (now a leaf once both children gone) in one pass: asserts tailA/tailB/preamble unmatched, the captured preamble node unreachable from root (upward collapse completed, no stranded childless internal node), the new request fully cached, final tokens==14<=budget, t.tokens==live Sigma edge lengths. All PASS.
